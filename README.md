# BackOffice Project

The BackOffice project is meant to be an exploratory to prove out the following technologies in a cloud environment.

* [akka](http://akka.io)
* [Casbah](http://api.mongodb.org/scala/casbah/2.0/)
* [mongoDB](http://www.mongodb.org)
* [Salat](https://github.com/novus/salat)
* [Simple Build Tool (SBT)](https://github.com/harrah/xsbt/wiki/)
* [Scala](http://www.scala-lang.org)
* [ScalaTest](http://scalatest.org)
* [Scalatra](http://www.scalatra.org)
* [specs2](http://etorreborre.github.com/specs2/)
* [Spray](http://spray.github.com/latest/introduction/)
* [subcut](https://github.com/dickwall/subcut)
